Token kaise banayein? (ek dafa ka kaam)
  1. GitHub par login karein → apni profile picture pe tap karein → Settings
  2. Sabse neeche left sidebar mein Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Naam de dein (e.g. pigeon-admin), Expiration set kar lein (e.g. 90 days)
  5. Repository access → "Only select repositories" → apni repo select karein
  6. Permissions → "Repository permissions" → Contents → "Read and write" select karein
  7. Generate token dabayein, jo token (github_pat_...) milay usay copy karein aur upar wale box mein paste kar dein
Ye token sirf isi browser mein save hota hai aur seedha GitHub ko jata hai — kahin aur nahi bhejta.
