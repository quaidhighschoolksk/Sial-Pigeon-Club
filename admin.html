🔐 Google Sign-In Setup

Ek dafa ka kaam — apna Google OAuth Client ID aur allowed Gmail accounts daal kar save karein.

Client ID kaise banayein?
  1. Google Cloud Console → Credentials par jayein (naya project bana lein agar nahi hai)
  2. OAuth consent screen configure karein — User type: External, apna Gmail add karein bas
  3. Create Credentials → OAuth client ID → Application type: Web application
  4. Authorized JavaScript origins mein apni site ka URL daalein (jahan yeh admin.html host hogi, e.g. https://username.github.io)
  5. Client ID copy karein, upar paste kar ke Save karein

Token kaise banayein? (ek dafa ka kaam)
  1. GitHub par login karein → profile picture → Settings
  2. Neeche Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Naam de dein, Expiration set karein
  5. Repository access → apni repo select karein
  6. Permissions → Contents → "Read and write"
  7. Token copy kar ke upar paste karein
